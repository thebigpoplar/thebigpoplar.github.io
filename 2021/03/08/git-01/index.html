git 基本常用命令

发表于2021-03-08|Git
|字数总计:264|阅读时长:1分钟|阅读量:
  1. git add [file]：向暂存区添加文件或文件夹。
  2. git commit -m “message”：提交暂存区的文件或文件夹到本地仓库并添加 commit message。
  3. git status：查看当前工作区、暂存区、本地仓库的文件状态。
  4. git log：查看本地仓库的提交日志。
  5. git checkout [branch]：切换到指定分支或者 commit。
  6. git branch：查看所有本地分支。
  7. git branch [branch]：创建新的分支。
  8. git merge [branch]：将指定分支合并到当前分支。
  9. git push：将本地分支推送到远程分支。
  10. git pull：从远程分支拉取到本地分支。
  11. git clone [url]：克隆远程仓库到本地。
  12. git remote add [name] [url]：添加远程仓库并命名。
  13. git stash：将当前工作区的修改暂存起来。
  14. git stash apply：恢复最近一次 stash 的修改。
  15. git diff：查看工作区和暂存区、本地仓库之间的文件差异。
  16. git reset [file]：取消暂存区的文件修改。
  17. git rm [file]：将文件从版本控制中删除。
  18. git tag [version]：给当前 commit 打上标签。
